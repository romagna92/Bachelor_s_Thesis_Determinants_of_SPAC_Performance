# Bachelor_s_Thesis_Determinants_of_SPAC_Performance
Code developed for the Bachelor's Thesis "Time-Series and Cross-Sectional Determinants of the Performance of Special Purpose Acquisition Companies"

Written in Wolfram (Mathematica), Data is included in the notebooks.

<pre>
  ├── Sample Descriptive Statistics            Descriptive Statistics of the SPAC-Sample
  ├── Cross-Sectional Analysis                 Regression Analysis to Identify Drivers of SPAC-Performance
  ├── Time-Series Analysis                     Examines Time-Varying Drivers of SPAC-Performance
  ├── Investment Strategies                    Examines a Variety of Long-, Short- and Long-short SPAC Investment Strategies 

  </pre>
